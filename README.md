# Make simple development environment in vagrant

* Required
```
	- install vagrant 4.3
	https://www.virtualbox.org/wiki/Download_Old_Builds_4_3
	http://download.virtualbox.org/virtualbox/4.3.34/VirtualBox-4.3.34-104062-OSX.dmg
	cf. vagrant box add ubuntu/trusty64
	
```

* Run
```
	bash build.sh
```	
	
* Flow
```
	1. nginx
		http://localhost:80
	2. jenkins
		http://localhost:8000
	3. mysql
	4. jira
		http://localhost:8080
	
```
